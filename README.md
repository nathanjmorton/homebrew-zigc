# homebrew-zigc

Homebrew tap for [zigc](https://github.com/nathanjmorton/zigc) — a C/C++ project & package manager powered by Zig's build system.

## Install

```sh
brew tap nathanjmorton/zigc
brew install zigc
```

## Upgrade

```sh
brew upgrade zigc
```

## Uninstall

```sh
brew uninstall zigc
brew untap nathanjmorton/zigc
```
